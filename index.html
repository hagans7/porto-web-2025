    Factors influencing work-life balance using confirmatory factor analysis using the R language

    ● This study aimed to identify the factors influencing work-life balance using Confirmatory Factor Analysis (CFA) based on existing theories.

    ● Data was collected from a survey evaluating work-life balance across 20 indicators and five latent variables: Healthy Body, Healthy Mind, Expertise, Connection, and Meaning, and CFA was performed using R to test two models.

    ● The analysis found that Model 2, which combined Healthy Body and Healthy Mind into a single latent variable, provided a better fit than Model 1, as indicated by lower BIC and higher chi-square values.

    ● Combining Healthy Body and Healthy Mind into a single latent variable resulted in a more robust model, demonstrating the close link between physical health and mental well-being in assessing work-life balance.


    Keyword: R language | Confirmatory Factor Analysis (CFA) | Data Analytic
